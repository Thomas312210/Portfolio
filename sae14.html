Se présenter sur Internet

Le projet : Site biographique sur Pedri González

Pour cette SAÉ, j'ai choisi de concevoir un site web dédié au footballeur du FC Barcelone, Pedri. La véritable plus-value de ce projet est que j'ai développé l'interface entièrement à la main, exclusivement en HTML5 et CSS3, sans utiliser d'intelligences artificielles.

Difficultés rencontrées & Solutions

1. L'adaptation sur Mobile (Responsive Design)

Créer une galerie d'images et un menu de navigation fluides sur ordinateur est une chose, mais faire en sorte que l'affichage reste lisible sur un smartphone en est une autre. Sans framework, j'ai dû gérer moi-même la réorganisation des blocs.

@media screen and (max-width: 768px) {
  header nav {
    display: flex;
    flex-direction: column;
    align-items: center;
  }
  .image-gallery, .photo-palmares {
    flex-direction: column;
  }
}
Extrait de mon fichier acceuil.css : Gestion du Responsive.
Ma solution : J'ai utilisé les Media Queries. Dès que l'écran passe sous la barre des 768px, la propriété flex-direction: column; force le menu de navigation et les galeries photos à s'empiler verticalement plutôt que horizontalement.

2. Créer une interface moderne sans framework

Je voulais un design attrayant, avec des effets de transparence sur le menu fixe et des animations au survol des images, sans alourdir le code avec du Javascript complexe.

header {
  background-color: rgba(30, 40, 60, 0.1);
  backdrop-filter: blur(4px);
  position: fixed;
  z-index: 1000;
}

@keyframes slide-bottom {
  0% { transform: translateY(-50px); opacity: 0; }
  100% { transform: translateY(0); opacity: 1; }
}
Extrait de mon code : Effet "Glassmorphism" et Animations CSS.
Ma solution : J'ai exploité des propriétés CSS avancées. Pour le menu, la propriété backdrop-filter: blur(4px) m'a permis de créer un effet "verre dépoli" très moderne (Glassmorphism). J'ai également intégré des règles @keyframes pour créer des animations fluides au survol de la photo de profil.

Bilan

Ce projet m'a prouvé qu'il est possible de créer un site professionnel et esthétique en maîtrisant uniquement les bases natives du web. Cela m'a donné des bases très solides sur le "Box Model" et le positionnement CSS, qui m'ont d'ailleurs servi pour concevoir ce portfolio !
